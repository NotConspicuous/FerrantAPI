For Developers.

Being able to archive backups of your projects in more than one place is often helpful, you never know when a bad actor strikes. Best to keep on the safe side ¯\_(ツ)_/¯.

This software is currently at α~0.1.1 and is under active development. There may be bugs.

It's FOSS.

We charge nothing, plus you can go change anything you want and self host it to make your own home-built experiance that is truely yours , who wouldn't want that?

We are not responsible for changes made to the program that are not by us.

Incredibly Fast.

Built on top of Fastify, this software is blazing fast. Perhaps not the fastest, but it's fast. We leverage the effeciency provided by our tools to make this software a great experiance. We even DDOSed this server during testing and it held up.

OAuth based.

We store no password. None. It is tied to your Github account. Using an OAuth Token, we provide a quick and painless way to sign in. If anything happened to the user tokens, we revoke all tokens.
